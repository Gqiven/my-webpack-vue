### my-webpack-vue
#从头构建一个vue组件的开发环境

### 目录结构

```bash
my-webpack-vue
| -build/     （存放构建配置文件）
  | -webpack.base.js  （公用配置）
  | -webpack.dev.js   （开发环境配置）
  | -webpack.prod.js  （生产环境配置）
| -config/    （存放构建配置的一些数据）
| -dist/       (存放构建的文件)
| -examples/  （存放组件示例c）
| -packages/  （存放组件）
| -src/       （存放公用js、样式、图片等文件）
```
